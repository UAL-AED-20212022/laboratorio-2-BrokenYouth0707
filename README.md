# laboratorio-2-BrokenYouth0707


RPI (novo_país) --> Registar um novo país no inicio de uma linked list

RPF (novo_país) --> Registar um novo país no fim da linked list

RPDE (novo_país) (país_indicado_dentro_da_lista) --> Adicionar um novo país depois do país indicado

RPAE (novo_país) (país_indicado_dentro_da_lista) --> Adicionar um novo país antes do país indicado

RPII (novo_país) (index) --> Adicionar um novo país indicando a posição onde vai ficar

VNE --> Mostrar o numero de países dentro da lista

VP (país) --> Verificar se o país indicado está dentro da lista

EPE --> Remove o primeiro país da lista

EUE --> Remove o ultimo país da lista

EP (país) --> Remove o país indicado da lista
